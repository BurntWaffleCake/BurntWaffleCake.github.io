Chi-Square(χ2) Testing

Used to test the counts of categorical data
3 Different Types:
Goodness of fit (GOF), used for univariate data
Independence used for bivariate data
Homogeniety used for bivariate data
Different degrees of freedom (df) hav different curves
skewed right, as df increases, curve shifts toward right and becomes more like a normal curve

Assumptions for Chi-Square Testing

Simple Random Sample
all expected counts >= 5 **Must list expected counts**

Chi-Squared Test Statistic Formula

To find p-value, use χ2cdf() on calculator. χ2 is your LB and 1E99 for UB. df depends on what type of χ2 you are doing.

Steps for χ2 Test

Step 1: State hypothesis (written words for χ2)
Step 2: Check assumptions (randomness and expected counts >= 5) write down name of test
Step 3: Calculations (χ2, df, p-value, α)
Step 4: Conclusion (same format as other tests, suggests that there is evidence that the Alternative Hypothesis is true or not, prompt can be worded towards null hypothesis so be careful)


χ2 Goodnes of Fit (GOF) Test

Uses univariate data
Used to see how well the observed counts "fit" what we expect the count to be
df = {# of categories} - 1

GOF Hypothesis

Ho: All proportions of {context} are equal to {expected proportions}
Ha: At least on proportion of {context} is not the same as {expected proportions}


Expected Counts and Degrees of freedom Formulas (Independence and Homogeniety)

expected counts = (row total * column total) / table total
df = (row # - 1)(column # - 1)

χ2 Test for Independence

Used with categorical, bivariate data
Used to see if two variables are associated (dependent) or not associated (independent)
Key phrasing: independent/dependent, associated, related to, affected by, etc.
Assumptions and formulas are same as GOF ^

Independence Hypothesis

Ho: {two variables} are independent
Ha: {two variables} are dependent


χ2 Test for Homogeniety

Used with bivariate categorical data
Used to see if two populations are the same (homogenous) or not
Key Phrasing: is different from a difference between, is the same, etc.
Everything is the same as an independence test except hypothesis

Homogeniety Hypothesis

Ho: All the proportions of {first variable} are the same for {second variable}
Ha: At least on of the proportions of {first variable} is different for {second variable}


Regression Inference with Computer Outputs

Degrees of freedom: df = n - 2 (Because there are two variables x and y)
Confidence Interval Equation: b ± t* (SEb) (slope ± t* * Standard Error of slope)

Hypothesis

Ho: β = 0 (where β is the population slope of ... {context})
Ha: β ≠ 0

Calculations

t = (b - β) / SEb
p-value = 2 * tcdf(LB, UB, d) (df = n - 2)
